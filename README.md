**Prerequisites**

    node v6.3.x
    npm  v3.10.x
 
**Initial setup**

    npm i
 
**clean dev build**

    npm run clean:build:dev
    
    or
    
    sh build-dev.sh
    
**clean build prod**

    npm run clean:build:prod
    
    or
    
    sh build-prod.sh

**start dev server**

    npm run dev:server
    
    or
    
    sh watch-run.sh
